## üìÇ Project Descriptions

### LEVEL 1: BEGINNER
* **Project 1: Personal Portfolio Website** A professional personal portfolio website created using HTML, CSS, and basic JavaScript. It includes sections such as "About Me," "Projects," and "Contact" to showcase my skills and journey.
* **Project 2: Responsive Landing Page** A landing page for a fictional product or service designed with a mobile-first approach. It uses CSS media queries to ensure the layout adjusts seamlessly across different screen sizes.

### LEVEL 2: INTERMEDIATE
* **Project 3: Interactive To-Do List Application** A web application that allows users to manage tasks effectively. Features include adding, editing, deleting, and marking tasks as complete, focusing on DOM manipulation and event handling.
* **Project 4: Basic Blog Application** A blogging platform where users can publish and view posts. This project involved setting up a backend server and performing basic CRUD operations to handle blog data.

### LEVEL 3: ADVANCED
* **Project 5: E-Commerce Website** A functional full-stack e-commerce store built using the MERN stack. It features user authentication, a dynamic shopping cart, and a product gallery to simulate a real-world retail experience.
* **Project 6: Real-Time Chat Application** A real-time communication platform where users can join rooms and exchange messages instantly. It utilizes WebSocket technology to manage live user sessions and message broadcasting.

## ‚öôÔ∏è Setup and Execution Instructions

### Prerequisites
* **Node.js** (v14 or higher) and **npm** installed on your machine.
* A modern web browser (Chrome, Firefox, or Edge).

### Level 1 Setup (Static Projects)
1.  Navigate to the specific project folder (e.g., `Level1/Project1`).
2.  Open the `index.html` file in your preferred web browser to view the site.

### Level 2 Setup (JavaScript & Backend)
1.  Navigate to the project folder (e.g., `Level2/project4`).
2.  Open your terminal and install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    node server.js
    ```
4.  Open `http://localhost:5000` in your browser.

### Level 3 Setup (MERN & WebSockets)

#### Project 5: E-Commerce Store
1.  **Backend Setup:**
    * Navigate to `level3/project 5/backend`.
    * Run `npm install` followed by `node server.js`.
2.  **Frontend Setup:**
    * Navigate to `level3/project 5/frontend`.
    * Run `npm install` followed by `npm start`.
    * The application will launch at `http://localhost:3000`.

#### Project 6: Chat Application
1.  Navigate to `level3/project6`.
2.  Install dependencies using `npm install`.
3.  Start the application with `node server.js`.
4.  Open `http://localhost:5000` in multiple browser tabs to test real-time chat functionality between users.
